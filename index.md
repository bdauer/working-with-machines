# A method for orchestrating complex work with AI agents

This collection describes a method built from practice — how to direct agents' attention, scope their context, run them in parallel, and recognize when the work has converged.

## Where to start

**[Quick start](quick-start.md)** — Concrete techniques to apply immediately. Start here if you want to change how you work today and read the theory later.

**[Method](method.md)** — The full framework: phases, knowledge accumulation, the human role, convergence. Start here if you want to understand why before how.

## The collection

Every page is self-contained and can be entered directly. Cross-references let you follow threads without a required reading order.

- **[Method](method.md)** — The operational core. Phases, design principles, the human role.
- **[Formation](formation.md)** — Why the method has its particular character. Optional background.
- **[Quick start](quick-start.md)** — Actionable techniques extracted from the full collection.
- **[Agent patterns](agent-patterns.md)** — Behavioral patterns in LLM agents and how the method works with them.

  - [Planning](workflows/planning.md) — Decomposition tasks
  - [Implementation](workflows/implementation.md) — Building software
  - [Review](workflows/review.md) — Evaluating artifacts
  - [Documentation](workflows/documentation.md) — Producing written artifacts
